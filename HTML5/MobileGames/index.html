Where are we?

  • 2004 - Canvas created by Apple

  • 2005 - Canvas integrated into Gecko

  • 2008 - Chrome released, JavaScript wars are on

  • Mar 2011 - IE9 released, finally supports canvas

  • Mar 2011 - WebGL "released" (On mobile only on Firefox, Opera)

  • Nov 2011 - iOS 5 released (GPU Accel. canvas)

  • Jun 2012 - Chrome for Android (GPU Accel. canvas)
